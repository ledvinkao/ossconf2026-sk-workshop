This repository relates to the workshop held at the OSSConf2026 conference (see https://ossconf.fri.uniza.sk/). An R script is crucial here, documenting what was the subject of the exercises regardin geodata processing in R. The processed data can be found at https://opendata.chmi.cz/, at https://www.perun-klima.cz/scenare/ or downloaded by using the functions of the {RCzechia} R package.
